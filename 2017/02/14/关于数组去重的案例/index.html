二、技术

第一篇是关于如何输出数组中最小的数字的方法。
首先定义了一个数组，数组中随便放了一些数字，然后定义了一个 num ，当然，这个 num 的值并不是固定的，任何一个数字都是可以的。
再接下来就是通过for循环去遍历数组了，在遍历数组的时候用定义的 num 与数组中的每一个数字进行比较，如果大于数组中的这个数字，那么就把 num 重新定义为这个数字，这样就能将数组中最小的那个数字输出了，代码如下：
var arr = [1,121,2,5,5,6,8,50,0,0]; var num = arr[0]; for(var j = 0; j < arr.length - 1; j++) { (function (a) { if(num > arr[a]) { num = arr[a]; } })(j) } console.log(num);
第二篇是关于如何找到字符串中出现次数最多的字母，并且将它出现的次数记录并输出的方法。
首先同样是定义了一个字符串 str ，然后又定义了一个空的对象 obj ，通过for循环将字符串中的字符依次输出，并且计算出他们出现的次数，然后通过for in利用上一篇比较大小的原理将二者输出即可，代码如下：
var str = 'aaabaabb'; var obj = {}; for(var i = 0; i < str.length; i++){ var k = str.charAt(i); if(obj[k]) { obj[k]++; }else { obj[k] = 1; } } var m=0; var i=null; for(var k in obj){ if(obj[k] > m){ m = obj[k]; i = k; } } console.log(i+':'+m);
